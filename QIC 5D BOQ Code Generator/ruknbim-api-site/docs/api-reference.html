⚙️ API Reference

Complete class and method documentation for the QicBoqMapper namespace

QicBoqManager

Static class containing the core business logic.

GetElements

public static List<Element> GetElements(
    Document doc,
    UIDocument uiDoc,
    string selectionMode,
    List<string> selectedCategoryNames,
    string categoryFilterText = "",
    string familyFilterText = "",
    string typeFilterText = ""
)

Collects Revit elements based on scope (Current Selection / Active View / Entire Model) and applies category, family, and type text filters.

CreateSharedParameters

public static void CreateSharedParameters(
    Document doc,
    List<string> categoryNames
)

Creates 14 shared instance parameters and binds them to all specified categories using a temporary shared parameter file.

PerformMapping

public static List<AuditRecord> PerformMapping(
    Document doc,
    List<Element> elements,
    List<BoqRecord> excelRecords,
    string separator,
    string matchingMethod,
    bool caseInsensitive,
    out int matchedCount,
    out int unmatchedCount
)

Matches elements to BOQ records using dual strategy (Element ID first, then Category+Family+Type). Returns audit records with status for each element.

UpdateParameters

public static void UpdateParameters(
    Document doc,
    List<Element> elements,
    List<AuditRecord> auditRecords,
    List<BoqRecord> excelRecords,
    string matchingMethod,
    bool caseInsensitive
)

Writes all mapped values (required + optional) to element parameters inside a Revit transaction. Supports flexible parameter name lookup with variants (underscore, space, title case).

QicBoqExcelService

Static class for reading Excel BOQ mapping files.

LoadBoqRecords

public static List<BoqRecord> LoadBoqRecords(string filePath)

Reads an Excel file and returns a list of BoqRecord objects. Auto-detects column headers and supports flexible naming.

QicBoqExportService

Static class for exporting Revit elements to Excel.

ExportElements

public static void ExportElements(
    string filePath,
    Document doc,
    List<Element> elements
)

Creates an Excel workbook with "Revit Elements" data sheet and empty "BOQ Mapping" template sheet.

Data Models

BoqRecord

Represents a row from the Excel BOQ mapping file.

public class BoqRecord
{
    // Matching fields
    string Category, FamilyName, TypeName, ElementId

    // Required BOQ fields
    string PackageNo, BillNo, SystemCode, PageNo, ItemNo

    // Optional fields
    string DistrictCode, AssetGroup, AssetType,
           LocationCode, Description,
           AbsL1, AbsL2, AbsL3
}

AuditRecord

Represents the mapping result for a single Revit element.

public class AuditRecord
{
    // Element identity
    string ElementId, UniqueId, Category,
           FamilyName, TypeName, Level,
           Workset, Mark

    // Mapped values
    string PackageNo, BillNo, SystemCode,
           PageNo, ItemNo, GeneratedBoqCode

    // Audit
    string Status    // "Success", "Warning", "Error"
    string Remarks   // Human-readable description
}

CategoryMappingService

Maps friendly category names to Revit BuiltInCategory enum values.

GetCategoryNames

public static IEnumerable<string> GetCategoryNames()

Returns all 40+ supported category display names.

GetBuiltInCategory

public static BuiltInCategory GetBuiltInCategory(string name)

Converts a display name to the corresponding BuiltInCategory enum value. Throws ArgumentException if not found.
