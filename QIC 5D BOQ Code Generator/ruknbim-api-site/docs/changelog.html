📋 Changelog

Version history and release notes

v1.0.0 — Initial Release

June 2025

🎉 Features

  • QIC 5D BOQ Manager window with dark theme UI
  • Export Revit elements to Excel (Revit Elements + BOQ Mapping sheets)
  • Import Excel BOQ mapping files with auto-detected headers
  • Dual matching engine: Element ID + Category/Family/Type
  • Validation mode (dry-run without writing data)
  • Generate 5D BOQ codes — creates 14 shared parameters
  • Audit report export (Excel, CSV, JSON)
  • 40+ Revit category support (Architecture, Structure, MEP, Site)
  • Scope selection: Current Selection, Active View, Full Model
  • Category/Family/Type text filters
  • Configurable separator styles (Dash, Dot, Underscore, Comma)
  • Supabase-backed license activation system
  • Auto-deploy MSBuild target
  • Inno Setup installer script

🔧 Technical

  • .NET Framework 4.8 targeting Revit 2023
  • WPF with MVVM pattern
  • EPPlus 5.8.14 for Excel operations
  • ExternalEvent pattern for Revit API thread safety
  • AssemblyResolve handler for dependency loading

Roadmap

  • 🔲 Multi-Revit version support (2024, 2025)
  • 🔲 Navisworks integration for WPH Export
  • 🔲 Batch processing across multiple models
  • 🔲 Cloud sync for BOQ mappings
  • 🔲 Visual mapping preview with element highlighting
  • 🔲 Undo/rollback for generated codes
