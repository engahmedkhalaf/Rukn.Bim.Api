🚀 Quick Start Guide

Get up and running with 5D BOQ code generation in 5 minutes

Step 1: Activate Your License

  1. Open Revit and navigate to the QIC Tools ribbon tab
  2. Click QIC 5D BOQ Manager — a license prompt will appear
  3. Enter your email and activation code
  4. Click Activate — the system validates against the server
💡 Tip

Your license is cached locally. You won't need to re-activate unless it expires or you switch machines.

Step 2: Export Your Model Elements

  1. Open a Revit project with model elements
  2. In the QIC 5D BOQ Manager window, select your scope:
    • Current Selection — only pre-selected elements
    • Active View — elements visible in the current view
    • Full Model — all model elements
  3. Optionally filter by Category, Family, or Type
  4. Click Export Elements and save the Excel file

The exported workbook contains two sheets:

  • Revit Elements — all element data (ID, Category, Family, Type, Level, etc.)
  • BOQ Mapping — an empty template for your BOQ codes

Step 3: Fill in BOQ Codes

Open the Excel file and fill in the BOQ Mapping sheet (or add columns to the Revit Elements sheet):

| Category | Family Name | Type Name   | Package No | Bill No | System Code | Page No | Item No |
|----------|------------|-------------|------------|---------|-------------|---------|---------|
| Walls    | Basic Wall | Generic 200 | PKG01      | B02     | ARC         | P05     | I001    |
| Doors    | Single     | 0900x2100   | PKG01      | B02     | ARC         | P07     | I015    |
ℹ️ Matching

Elements are matched by Element ID first (if column exists), then by Category + Family + Type combination.

Step 4: Import & Validate

  1. Click Import Excel in the Manager window
  2. Select your filled-in Excel file
  3. The status bar shows how many records were loaded
  4. Click Validate Mapping to preview results without writing data

Validation shows:

  • Matched — elements successfully linked to BOQ records
  • Warnings — matched but with missing field values
  • Errors — no mapping found for these element types

Step 5: Generate Codes

  1. Review the validation results
  2. Click Generate 5D BOQ Codes
  3. The add-in will:
    • Create all 14 shared parameters (if not existing)
    • Match each element to its BOQ record
    • Write parameter values to every matched element
    • Generate the composite QIC_5D_BOQ CODE
  4. A summary dialog shows the results
💡 Example Output

With separator set to Dash: PKG01-B02-ARC-P05-I001

Step 6: Export Audit Report

After generating codes, click Audit Report to export a full trail of every element processed. Available formats:

  • Excel (.xlsx) — full formatted report
  • CSV (.csv) — UTF-8 plain text
  • JSON (.json) — machine-readable format
