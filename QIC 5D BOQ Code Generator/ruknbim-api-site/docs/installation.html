📦 Installation Guide

Step-by-step setup for QIC 5D BOQ Code Generator on Revit 2023+

Prerequisites

  • Autodesk Revit 2023 or later installed
  • Windows 10/11 (64-bit)
  • .NET Framework 4.8 (included with Windows 10 1903+)
  • Administrator access for initial setup

Option 1: Installer (Recommended)

  1. Download the latest QicBoqMapperSetup.exe from the releases page
  2. Run the installer — it will auto-detect your Revit installation
  3. Follow the wizard to completion
  4. Launch Revit — the "QIC Tools" tab will appear in the ribbon
💡 Tip

The installer copies files to C:\ProgramData\Autodesk\Revit\Addins\2023\ automatically.

Option 2: Manual Installation

  1. Build the solution or download the compiled DLLs
  2. Copy QicBoqMapper.addin to: 
    C:\ProgramData\Autodesk\Revit\Addins\2023\
  3. Copy all DLLs to: 
    C:\ProgramData\Autodesk\Revit\Addins\2023\QicBoqMapper\
  4. Copy the Resources folder to: 
    C:\ProgramData\Autodesk\Revit\Addins\2023\QicBoqMapper\Resources\

Required Files

Addins\2023\
├── QicBoqMapper.addin
└── QicBoqMapper\
    ├── QicBoqMapper.dll
    ├── EPPlus.dll
    ├── Microsoft.IO.RecyclableMemoryStream.dll
    └── Resources\
        ├── QicBoqManager_16.png
        ├── QicBoqManager_32.png
        ├── License_16.png
        ├── License_32.png
        ├── About_16.png
        └── About_32.png

Option 3: Developer Build

If you're building from source:

dotnet build QicBoqMapper.csproj -c Release
ℹ️ Note

The .csproj includes an AutoDeployToRevit MSBuild target that copies everything to the Revit Addins folder automatically after each build.

Verify Installation

  1. Launch Autodesk Revit 2023
  2. Look for the "QIC Tools" tab in the ribbon
  3. You should see 3 buttons: Manager, License, About
  4. Click About to verify version information

Troubleshooting

Add-in doesn't appear in ribbon

  • Verify the .addin file is in the correct folder
  • Check that the Assembly path in the .addin file points to the correct DLL location
  • Look in Revit's journal file for error messages

DLL loading errors

  • Ensure EPPlus.dll is in the same folder as QicBoqMapper.dll
  • The add-in includes an AssemblyResolve handler to load dependencies from its own folder
⚠️ Important

Close Revit completely before installing or updating add-in files. Revit locks DLLs while running.
