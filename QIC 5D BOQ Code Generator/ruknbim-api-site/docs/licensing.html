🔑 Licensing

Activation, renewal, and license management

How Licensing Works

RUKN BIM add-ins use a cloud-validated license system:

  1. User receives an email and activation code upon purchase
  2. The add-in validates credentials against the Supabase backend
  3. On success, the license is cached locally in the Windows Registry
  4. Subsequent launches check the local cache — no internet required unless expired

Activation Flow

  1. Click any tool button in the QIC Tools ribbon tab
  2. If no valid license is found, the License Window appears
  3. Enter your registered email and activation code
  4. Click Activate
  5. The system validates with the server and stores the result
ℹ️ Offline Support

After initial activation, the add-in works offline until the license expiration date. No internet connection is needed for day-to-day use.

License Storage

License data is stored in the Windows Registry:

HKEY_CURRENT_USER\Software\QicTools\QicBoqMapper\
├── Email           (string)
├── ActivationCode  (string)
├── LastVerified    (ISO 8601 datetime)
└── ExpiresAt       (ISO 8601 datetime)

Expiration & Renewal

  • The ExpiresAt timestamp is returned by the server during activation
  • When expired, the add-in prompts re-activation
  • If the server returns an active license with a new expiry, it updates locally
  • Contact your administrator to renew a subscription before expiry
⚠️ Expired License

An expired license blocks all commands except the License and About buttons. A dialog will inform you that renewal is needed.

Validation Rules

  • Email must be a valid email format
  • Activation code must be at least 4 characters
  • Server checks: email + code match in licenses table, status = "active"
  • If expires_at is past, activation is denied

Troubleshooting

"Supabase connection failed"

  • Check your internet connection
  • Verify firewall isn't blocking HTTPS requests
  • Contact support if the issue persists

"License Expired" dialog on startup

  • Your subscription has ended
  • Contact the administrator for a renewal code
  • Click License in the ribbon to re-activate with new credentials

Moving to a new machine

  • Simply re-activate with the same email and code on the new machine
  • The license is tied to the email/code pair, not the machine
